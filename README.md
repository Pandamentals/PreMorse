# PreMorse

> Stop forgetting to ask the obvious.

Describe your situation before any call, appointment, or meeting — PreMorse generates the questions you'll wish you asked but always forget in the moment.

## What it's for

- 📞 Phone calls
- 🏥 Doctor & specialist appointments
- 👶 Kids appointments
- 🏫 School & daycare meetings
- 💼 Work meetings
- ⚖️ Legal & financial
- 🔧 Service & repair
- 🏠 Landlord & housing

## How it works

1. Tap the situation type
2. Describe what's going on
3. Get a categorized checklist of questions to ask
4. Check them off as you go

## Setup

Hosted via GitHub Pages — open the URL in Safari, enter your [Anthropic API key](https://console.anthropic.com) once, and add to your home screen.

Your API key is stored locally on your device only. It is never in the code.

## Install on iPhone

Safari → Share → **Add to Home Screen**
